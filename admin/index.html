⚙️ 구글 시트 직접 수집 설정

1

구글 시트 새로 만들기

sheets.google.com에서 새 시트 생성.
시트 이름은 자유롭게 설정하세요 (아기도구함 Analytics 등).

2

Apps Script에 tracker.gs 코드 배포

시트 상단 메뉴 → 확장 프로그램 → Apps Script
admin/tracker.gs 파일 내용을 Code.gs에 붙여넣기
→ 배포 → 새 배포 → 웹 앱
→ 실행 계정: 나 / 액세스: 모든 사용자
→ 배포 URL 복사

3

config.js에 TRACKER_URL 입력

TRACKER_URL: 'https://script.google.com/macros/s/XXXX/exec',
4

시트 자동 생성 확인

사이트에 방문하면 구글 시트에 아래 4개 탭이 자동 생성돼요:

pageviews     — 페이지뷰 로그 (방문자·기기·UTM 등)
events        — 이벤트 로그 (클릭·스크롤·검색 등)
daily_summary — 일별 집계 (페이지뷰·툴사용·수익추정)
tool_stats    — 툴별 누적 사용 통계
5

Admin 대시보드 연결

아래 입력란에 TRACKER_URL을 입력하면 이 대시보드에서 바로 데이터를 불러올 수 있어요.
(config.js에 이미 입력했다면 자동 인식됩니다)
